StdService
    只负责：查 SNOMED
    输入：一个 query
    输出：原始结构化候选

MedicalRetriever
    只负责：把 SNOMED 结果包装成 RAG docs
    输入：一个 query
    输出：page_content + metadata

MedicalStandardizer
    只负责：处理一整段文本
    输入：一段医疗文本
    输出：多个实体 + 每个实体对应的候选术语

StdService：我给你一个医学词，你帮我查 SNOMED

MedicalRetriever：我给你一个 query，你帮我返回 RAG 能用的 docs

MedicalStandardizer：我给你一整段病历，你先抽实体，再逐个查 SNOMED，最后按实体整理结果
