#### 改动 3 — `abbr_service.py` 新增 `_build_expanded_text_deterministic()`

放在 `_rebuild_expanded_text`(229–241)**下方**,**不要删** `_rebuild_expanded_text`(它仍被 MappingSupportVerifier 实验分支引用)。新函数用 token 边界正则、按出现位置从后往前替换,避免 offset 错位与子串误伤(`CP` 不命中 `CPR`):

```python
import re  # 文件顶部已 import json;补 import re

def _build_expanded_text_deterministic(self, text: str, chosen: list[dict]) -> str:
    """确定性扩写:对每个 {abbreviation -> expansion} 按 token 边界替换。
    - \\b...\\b 保证不误伤子串(CP 不命中 CPR)
    - 从后往前替,避免多次替换的 offset 错位
    - 只替换 chosen 里有 expansion 的项;否定/其它词原样保留
    """
    if not chosen:
        return text

    # 收集所有匹配区间 (start, end, expansion),再按 start 从后往前替
    spans = []
    for item in chosen:
        abbr = item.get("abbreviation")
        expansion = item.get("expansion")
        if not abbr or not expansion:
            continue
        pattern = re.compile(rf"\b{re.escape(abbr)}\b")
        for m in pattern.finditer(text):
            spans.append((m.start(), m.end(), expansion))

    # 从后往前替,后面的替换不影响前面的索引
    spans.sort(key=lambda s: s[0], reverse=True)
    result = text
    for start, end, expansion in spans:
        result = result[:start] + expansion + result[end:]
    return result
```
> 大小写:gate 里 `abbr` 已 `.upper()`,但原句里缩写也通常是大写(gate 只放行已知缩写或全大写 token,见 `_should_consider_abbreviation` 622–624)。`re.escape(abbr)` 用大写 abbr 对原句匹配,默认大小写敏感**正好**——只替大写的 `CP`,不碰小写词。若担心 `htn` 这类小写已知缩写,可加 `re.IGNORECASE`,但**默认先不加**,benchmark 验证后再说。

## 批次 2 · mapping 粒度状态机 + 失败隔离 + 增量重算(核心 ⑤)

> 把工作单位从「整句」降到「单个 mapping」:每个 mapping 走 `PENDING → 检索 → verify → LOCKED_OK / LOCKED_ABSTAIN`;失败只隔离重修自己,LOCKED_OK 冻结复用;reflect 从「整句重写」改为「从候选池换一个未试候选」。**依赖批次 1 的确定性替换**(隔离的物理前提)。

### 给 Codex 的指令

**改前先 Read `abbr_service.py · expand_verify_with_retry`(362–512)和 `abbr_reflection_service.py`。**

重构 `expand_verify_with_retry` 的循环(362–512)。新结构(伪代码,Codex 按此落地,保留 `attempts` 留痕):

1. **建 mapping 状态对象**(召回后、进循环前)。从批次 1 的 `candidate_infos` 给每个 chosen mapping 带:
   ```python
   # 每个 mapping 一个状态机条目
   m = {
       "abbreviation": ...,
       "expansion": ...,           # 当前选中的
       "label": ...,
       "status": "PENDING",        # PENDING / LOCKED_OK / LOCKED_ABSTAIN
       "pool": [c["expansion"] for c in info["candidates"]],  # 候选池(全部)
       "tried": {best},            # 已试集合,初始含当前选中
       "std_cache": None,          # 检索结果缓存(增量复用)
       "changed": True,            # 本轮 expansion 是否变过(初始 True → 要检索)
   }
   ```
2. **循环改为 while + 兜底次数**:
   ```python
   for attempt_index in range(max_retries + 1):
       # 只处理未锁定的 mapping
       pending = [m for m in states if m["status"] == "PENDING"]
       if not pending:
           break
       for m in pending:
           if m["changed"]:                       # 增量:只对变过的重检索
               m["std_cache"] = retrieve(m["expansion"], ...)
               m["changed"] = False
           # 用各 mapping 自己的 std_cache 攒 mapping_standardizations
       verification = verify_mappings(text, current_expanded_text, mapping_standardizations)
       # 按 abbreviation 把每条 mapping_validation 对回 m
       for m in pending:
           mv = 找到 verification 里 abbreviation==m["abbreviation"] 的那条
           if mv 通过:
               m["status"] = "LOCKED_OK"
           else:
               untried = [e for e in m["pool"] if e not in m["tried"]]
               if untried:
                   new_e = reflect_select(m, verification)   # 选择型 reflect,见改动B
                   m["expansion"] = new_e; m["tried"].add(new_e); m["changed"] = True
               else:
                   m["status"] = "LOCKED_ABSTAIN"
       # 有任何 mapping 变过 → 重拼整句(确定性替换)
       current_expanded_text = self._build_expanded_text_deterministic(text, [可见的 LOCKED_OK + PENDING mapping])
       attempts.append(本轮快照)
       # 继续条件:还有 PENDING 且其 untried 非空才转;否则下一轮 pending 为空会 break
   ```
3. **检索增量**:`m["changed"]==False` 的 mapping(已 LOCKED_OK 或本轮没换词)**跳过检索**,复用 `m["std_cache"]`。
4. **继续条件**:循环靠「还有没有 PENDING mapping」驱动(第 2 步开头 `if not pending: break`);`max_retries` 只作兜底上限。
5. **出口**:`final_result` 汇总每个 mapping 的终态(`status`)。`LOCKED_ABSTAIN` 的 mapping——**决定它在 `expanded_text` 里是否回退为原缩写**:本批策略是「ABSTAIN 的 expansion 不替换、保留原缩写」,即重拼整句时只对非 ABSTAIN 的 mapping 替换。
6. `attempts` 每轮快照保留(调试 + benchmark 归因)。

```python
def is_medical(self, text: str):
    """对孤立短语判是否医学实体 + 返回主 label。
    返回 (is_medical: bool, label: str|None, score: float)
    """
    ents = self.extract_entities(text)
    if not ents:

    # 取分数最高的实体的 label
    top = max(ents, key=lambda e: e["score"])
```
